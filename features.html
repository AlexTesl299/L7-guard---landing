Технические спецификации

⚙️

Архитектура и Маршрутизация

  • Go-powered Single Binary меньшн 10 МБ Весь бэкенд написан на Go, весит меньше 10 мегабайт и работает молниеносно.
  • Browser Extension (Clientless): Доступ работает через легкое браузерное расширение. Никаких десктопных VPN.
  • Zero-copy & SNI-routing: Высокоскоростная маршрутизация без "вскрытия" и расшифровки пакетов (No MITM).
  • Per-Tab Split Tunneling: Направляйте трафик в разные шлюзы прямо из соседних вкладок браузера.
  • Dynamic Domain Rotation: Умный обход кэша Chromium за счет автоматической ротации Wildcard-доменов.
  • Stealth Mode: Шлюз отдает 404 случайным ботам, прикидываясь несуществующей страницей.
🔐

Управление доступом

  • JIT Access: Временные доступы подрядчикам.
  • Stateful TTL: Изменение времени "жизни" токена на лету.
  • Instant Kill-Switch: Отзыв прав за миллисекунды без рестарта.
  • Bandwidth Quota: Жесткие лимиты на скачивание трафика.
  • Dynamic ACL: Изменение политик в реальном времени.
👻

WordPress Плагин

  • IP Whitelisting Core: Блокировка входа для всех IP, кроме шлюзов L7.
  • Dynamic API Updates: Авто-обновление списка разрешенных адресов.
  • Fail-safe: Отключение защиты одной строкой в wp-config.php.

  • Passphrase Tokens: Дополнительная защита токенов паролем.
